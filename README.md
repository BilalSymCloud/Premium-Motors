# Premium Motors — Car Dealership

A Next.js 14 (App Router) car dealership demo with TypeScript and Tailwind CSS.

## Setup

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Features

- **Home:** Grid of 4 cars with “Add to Cart”
- **Cart:** Managed via React Context
- **Checkout:** View selected cars and total price
- **Search:** Search bar (filtering to be implemented)

## Stack

- Next.js 14 (App Router)
- TypeScript
- Tailwind CSS
- Lucide React (icons)
